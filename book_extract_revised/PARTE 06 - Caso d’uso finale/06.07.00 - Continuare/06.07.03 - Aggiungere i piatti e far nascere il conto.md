# 06.07.03 - Aggiungere i piatti e far nascere il conto

## Qui il progetto comincia a diventare davvero una cassa

Finora abbiamo costruito:
- una finestra che compila;
- un elenco tavoli;
- un contesto selezionabile.

Adesso introduciamo il passaggio che cambia davvero la natura del progetto:
l'ordine nasce quando l'utente può aggiungere piatti al tavolo corrente.

Questo è il momento in cui la schermata smette di essere solo “una vista di tavoli” e diventa veramente una cassa.

## L'obiettivo di questo passo

Vogliamo ottenere:
- un catalogo di piatti;
- una collection di righe conto per ogni tavolo;
- un primo comando di aggiunta;
- la comparsa effettiva del conto corrente.

## Introduciamo `Piatto`

```csharp
using System.ComponentModel;
using System.Runtime.CompilerServices;

namespace CassaRistorante.Models
{
    public class Piatto : INotifyPropertyChanged
    {
        private int _id;
        private string _nome = "";

        private string _categoria = "";

        public int Id
        {
            get => _id;
            set

                    _id = value;

                }
            }
        }

        public string Nome
        {
            get => _nome;
            set
            {
                if (_nome != value)
                {
                    _nome = value;
                    OnPropertyChanged();
                }
            }
        }

        public decimal Prezzo
        {
            get => _prezzo;
            set
            {
                if (_prezzo != value)
                {
                    _prezzo = value;
                    OnPropertyChanged();
                }
            }
        }

        public string Categoria
        {
            get => _categoria;
            set
            {
                if (_categoria != value)
                {
                    _categoria = value;
                    OnPropertyChanged();
                }
            }
        }

        public event PropertyChangedEventHandler? PropertyChanged;

        protected void OnPropertyChanged([CallerMemberName] string? propertyName = null)
        {
            PropertyChanged?.Invoke(this, new PropertyChangedEventArgs(propertyName));
        }
    }
}
```

## Introduciamo `RigaConto`

```csharp
using System.ComponentModel;
using System.Runtime.CompilerServices;

namespace CassaRistorante.Models
{
    public class RigaConto : INotifyPropertyChanged
    {
        private Piatto _piatto = new();
        private int _quantita = 1;

        public Piatto Piatto
        {
            get => _piatto;
            set
            {
                if (_piatto != value)
                {
                    _piatto = value;
                    OnPropertyChanged();
                    OnPropertyChanged(nameof(NomePiatto));
                    OnPropertyChanged(nameof(PrezzoUnitario));
                    OnPropertyChanged(nameof(Subtotale));
                }
            }
        }

        public int Quantita
        {
            get => _quantita;
            set
            {
                if (_quantita != value)
                {
                    _quantita = value;
                    OnPropertyChanged();
                    OnPropertyChanged(nameof(Subtotale));
                }
            }
        }

        public string NomePiatto => Piatto.Nome;
        public decimal PrezzoUnitario => Piatto.Prezzo;
        public decimal Subtotale => PrezzoUnitario * Quantita;

        public event PropertyChangedEventHandler? PropertyChanged;

        protected void OnPropertyChanged([CallerMemberName] string? propertyName = null)
        {
            PropertyChanged?.Invoke(this, new PropertyChangedEventArgs(propertyName));
        }
    }
}
```

## Il modello `Tavolo` ora contiene il conto

```csharp
using System.Collections.ObjectModel;
using System.Collections.Specialized;
using System.ComponentModel;
using System.Linq;
using System.Runtime.CompilerServices;

namespace CassaRistorante.Models
{
    public class Tavolo : INotifyPropertyChanged
    {
        private int _id;
        private string _nome = "";
        private ObservableCollection<RigaConto> _righeConto = new();

        public Tavolo()
        {
            _righeConto.CollectionChanged += RigheConto_CollectionChanged;
        }

        public int Id
        {
            get => _id;
            set
            {
                if (_id != value)
                {
                    _id = value;
                    OnPropertyChanged();
                }
            }
        }

        public string Nome
        {
            get => _nome;
            set
            {
                if (_nome != value)
                {
                    _nome = value;
                    OnPropertyChanged();
                }
            }
        }

        public ObservableCollection<RigaConto> RigheConto
        {
            get => _righeConto;
            set
            {
                if (_righeConto != value)
                {
                    _righeConto.CollectionChanged -= RigheConto_CollectionChanged;
                    _righeConto = value;
                    _righeConto.CollectionChanged += RigheConto_CollectionChanged;
                    OnPropertyChanged();
                    OnPropertyChanged(nameof(IsOccupato));
                    OnPropertyChanged(nameof(Totale));
                }
            }
        }

        public bool IsOccupato => RigheConto.Any();

        public string StatoDescrittivo => IsOccupato ? "Occupato" : "Libero";

        public decimal Totale => RigheConto.Sum(r => r.Subtotale);

        private void RigheConto_CollectionChanged(object? sender, NotifyCollectionChangedEventArgs e)
        {
            OnPropertyChanged(nameof(IsOccupato));
            OnPropertyChanged(nameof(StatoDescrittivo));
            OnPropertyChanged(nameof(Totale));
        }

        public event PropertyChangedEventHandler? PropertyChanged;

        protected void OnPropertyChanged([CallerMemberName] string? propertyName = null)
        {
            PropertyChanged?.Invoke(this, new PropertyChangedEventArgs(propertyName));
        }
    }
}
```

## Il `ViewModel` si arricchisce

Ora abbiamo:
- elenco piatti;
- piatto selezionato;
- righe del conto corrente.

```csharp
using System.Collections.ObjectModel;
using System.ComponentModel;
using System.Linq;
using System.Runtime.CompilerServices;
using CassaRistorante.Models;

namespace CassaRistorante.ViewModels
{
    public class CassaRistoranteViewModel : INotifyPropertyChanged
    {
        private Tavolo? _tavoloSelezionato;
        private Piatto? _piattoSelezionato;

        public ObservableCollection<Tavolo> Tavoli { get; } = new();
        public ObservableCollection<Piatto> Piatti { get; } = new();

        public Tavolo? TavoloSelezionato
        {
            get => _tavoloSelezionato;
            set
            {
                if (_tavoloSelezionato != value)
                {
                    _tavoloSelezionato = value;
                    OnPropertyChanged();
                    OnPropertyChanged(nameof(RigheContoCorrenti));
                    OnPropertyChanged(nameof(TotaleCorrente));
                }
            }
        }

        public Piatto? PiattoSelezionato
        {
            get => _piattoSelezionato;
            set
            {
                if (_piattoSelezionato != value)
                {
                    _piattoSelezionato = value;
                    OnPropertyChanged();
                }
            }
        }

        public ObservableCollection<RigaConto> RigheContoCorrenti
            => TavoloSelezionato?.RigheConto ?? new ObservableCollection<RigaConto>();

        public decimal TotaleCorrente
            => TavoloSelezionato?.RigheConto.Sum(r => r.Subtotale) ?? 0m;

        public CassaRistoranteViewModel()
        {
            Piatti.Add(new Piatto { Id = 1, Nome = "Margherita", Prezzo = 7.50m, Categoria = "Pizze" });
            Piatti.Add(new Piatto { Id = 2, Nome = "Carbonara", Prezzo = 9.00m, Categoria = "Primi" });
            Piatti.Add(new Piatto { Id = 3, Nome = "Acqua", Prezzo = 2.00m, Categoria = "Bevande" });

            Tavoli.Add(new Tavolo { Id = 1, Nome = "Tavolo 1" });
            Tavoli.Add(new Tavolo { Id = 2, Nome = "Tavolo 2" });
            Tavoli.Add(new Tavolo { Id = 3, Nome = "Tavolo 3" });

            TavoloSelezionato = Tavoli.FirstOrDefault();
        }

        public event PropertyChangedEventHandler? PropertyChanged;

        protected void OnPropertyChanged([CallerMemberName] string? propertyName = null)
        {
            PropertyChanged?.Invoke(this, new PropertyChangedEventArgs(propertyName));
        }
    }
}
```

## Il primo comando di aggiunta

Qui introduciamo finalmente una vera azione.

```csharp
public ICommand AggiungiPiattoCommand { get; }

public CassaRistoranteViewModel()
{
    // ... seed dati

    AggiungiPiattoCommand = new RelayCommand(
        _ => AggiungiPiatto(),
        _ => TavoloSelezionato is not null && PiattoSelezionato is not null);
}

private void AggiungiPiatto()
{
    if (TavoloSelezionato is null || PiattoSelezionato is null)
        return;

    TavoloSelezionato.RigheConto.Add(new RigaConto
    {
        Piatto = PiattoSelezionato,
        Quantita = 1
    });

    OnPropertyChanged(nameof(RigheContoCorrenti));
    OnPropertyChanged(nameof(TotaleCorrente));
}
```

## Perché questo passaggio è così importante

Perché da qui in poi il progetto ha già il nucleo vero di una cassa:
- un tavolo attivo;
- un catalogo piatti;
- un conto che nasce davvero;
- un totale che cambia;
- una prima azione operativa.

È il punto in cui il caso finale smette di sembrare un esempio teorico.

## Regola pratica

Appena i piatti possono essere aggiunti al tavolo corrente e il conto compare davvero nella UI, il progetto entra nella sua fase viva.
Da qui in poi ogni miglioramento non cambia più l’identità dell’app: la rende solo più matura.
