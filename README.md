# WPF in pratica: dalle fondamenta a un progetto desktop vero

Questo repository raccoglie la versione completa del libro dedicato a WPF, pensato per accompagnare il lettore da una comprensione iniziale del framework fino alla costruzione di una piccola applicazione desktop credibile, leggibile e strutturata bene. Il percorso non si limita a spiegare controlli, binding o XAML come argomenti isolati, ma prova a tenere sempre insieme architettura mentale, esempi concreti, errori frequenti e scelte progettuali sensate, in modo che WPF emerga per quello che è davvero: non una semplice raccolta di tecniche, ma un sistema coerente di presentazione, dati, layout e comportamento.

Il materiale è organizzato in file Markdown separati per capitolo e sottocapitolo, così da permettere sia una lettura lineare sia una consultazione rapida dei singoli temi. Nelle sezioni che seguono trovi un accesso ordinato all’intero percorso del libro, con le parti principali, i capitoli intermedi e i link diretti ai file `.md`, in modo da poter entrare subito nel punto che ti serve senza perdere la visione d’insieme.
